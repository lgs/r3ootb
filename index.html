r3ootb by lgs

Rails 3 out of the box application templates

Rails 3 out of the box application templates (R3OOTB), actually runs Rails version 3.0.0.beta3 and is packed up with fine jewels: devise authentication blueprint css layout, built with Boks

Dependencies

Gems included by the bundle: * abstract (1.0.0) * actionmailer (3.0.0.beta3) * actionpack (3.0.0.beta3) * activemodel (3.0.0.beta3) * activerecord (3.0.0.beta3) * activeresource (3.0.0.beta3) * activesupport (3.0.0.beta3) * arel (0.3.3) * builder (2.1.2) * bundler (0.9.24) * devise (1.1.rc1) * erubis (2.6.5) * i18n (0.3.7) * mail (2.2.0) * memcache-client (1.8.2) * mime-types (1.16) * polyglot (0.3.1) * rack (1.1.0 c6805f) * rack-mount (0.6.3) * rack-test (0.5.3) * rails (3.0.0.beta3 43e2fd) * railties (3.0.0.beta3) * rake (0.8.7) * sqlite3-ruby (1.2.5) * text-format (1.0.0) * text-hyphen (1.0.0) * thor (0.13.4) * treetop (1.4.5) * tzinfo (0.3.20) * warden (0.10.3)
